# EMS Frontend

The Employee Management System frontend uses React, TypeScript, Vite, and
Material UI.

## Source structure

```text
src/
├── components/    # All React components
├── index.css      # Minimal global styles
├── main.tsx       # Application entry point
└── theme.ts       # Shared Material UI theme
```

New React components must be added under `src/components`. Feature-specific
subfolders can be introduced there as the application grows.

## Commands

```bash
npm run dev
npm run lint
npm run build
```
